# HerBody — Claims Matrix

**Rule:** every marketing sentence on any HerBody surface must trace to a row here. If it isn't here, it doesn't publish. Authorised claims must stay within GB NHC register wording (flexibility of wording is allowed only where the meaning to the consumer is the same and not stronger). Register entries change — the pre-publish checklist (`docs/COMPLIANCE_CHECKLIST.md`) requires a fresh lookup.

## Working notes

- **Wording flexibility:** ASA/Trading Standards accept rewording of authorised claims only where the meaning is the same for the average consumer and **no stronger**. When in doubt, use register wording verbatim. Every shortened ad form above has its verbatim parent on the landing page.
- **General health references** ("wellness", "feel like yourself") are non-specific benefits — permitted only when **accompanied by** a specific authorised claim on the same surface (retained Art. 10(3) rule). The compliance gate checks for a paired authorised claim.
- **Stacking risk:** placing row 4 (inositol) near row 11's B6 hormonal-activity claim creates an implied hormone-outcome message. Keep them on separate page sections; never in the same ad.
