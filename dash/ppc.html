Kill switch — if spend ever looks wrong

  1. Fastest: pause the campaigns directly in the platform UI (Google Ads / Meta Ads Manager / TikTok Ads). Drafts created by the fleet start paused; nothing runs that you haven't approved.

  3. Disable the agent-fleet workflow on the repo's Actions tab to stop all agent runs entirely.
