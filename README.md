# HerBody — website

Marketing site for HerBody, a halal clinical beauty-supplement house from London.
One clinical-dose pouch (creatine + Verisol® collagen + multivitamin + botanicals),
with £1 + 10% pledged to a woman the customer names.

Static HTML/CSS/JS — no build step. Just open `index.html`, or visit the live site.

## Live site

Published with GitHub Pages: **https://faizanshabbir777-cmd.github.io/herbody-site/**
